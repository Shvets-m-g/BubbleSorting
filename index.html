    Правила использования

    Программа разработана в качестве тестового задания для наглядной демонстрации сортировки массива методом Пузырька.

    Для запуска сортировки необходимо нажать на кнопку "СТАРТ". Во время работы программы кнопки отключены.

    Для генерации массива определенного размера необходимо ввести размер в поле "Размер" и нажать на кнопку "Сгенерировать". Затем для запуска сортировки нажмите на кнопку "СТАРТ".
